Web Security Projects

🔐 DVWA Web Application Security Testing

🔗 View Full Project on GitHub

Complete security testing project performed on DVWA using manual and automated techniques. This project includes SQL Injection, Authentication Security Testing, and Reflected XSS vulnerability analysis.

Tools Used: DVWA, XAMPP, SQLMap, OWASP ZAP, Chrome DevTools

🔥 SQL Injection Testing

Performed manual SQL Injection and automated SQLMap exploitation against DVWA running on XAMPP.

  • Authentication bypass
  • Database extraction
  • Users table dump
  • SQLMap automation

Key Findings: SQL Injection confirmed, database compromised, sensitive user data exposed.

🔥 Authentication Security Testing

Manual authentication and session management testing performed using browser developer tools.

  • Weak credentials detected
  • No account lockout
  • Unlimited login attempts
  • Broken session management
  • Insecure cookies

Session fixation and brute-force weaknesses identified.

🔥 Reflected XSS Testing

Tested DVWA Reflected XSS module manually and using OWASP ZAP.

  • Payload injection
  • HTML reflection analysis
  • ZAP active scan
  • High-risk XSS detection

Input was reflected without sanitization, allowing JavaScript execution.

🛡 Skills Demonstrated

  • Web Application Security Testing
  • SQL Injection Exploitation
  • Authentication Testing
  • Session Security Analysis
  • Cross-Site Scripting (XSS)
  • OWASP ZAP
  • SQLMap

⚠ Disclaimer

All testing was performed on DVWA, a purposely vulnerable training environment. These techniques were used strictly for ethical learning purposes.
